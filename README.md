# MEALFINDER 프로젝트

- 음식을 서치하고, 서치한 음식을 토대로 기호 음식을 추천받는 APP

# 사용스택

### FRONT_END

- REACT_NATIVE, REDUX-TOOLKIT, TYPESCRIPT, KOTLIN

### BACK_END

- EXPRESS, POSTGRE_SQL

# 기능구현

- 회원가입 탈퇴
- 로그인(일반로그인, 카카오톡로그인)
- 자신의 프로필 수정(닉네임)
- 일반 음식 서치 && 분류탭을 기준으로 서치
- 다크모드
